# ✦ BenDown 笨蛋阅读器

> 一款简洁纯粹的 Markdown 阅读器，专注阅读体验，让文字不被打扰。

---

## 📌 产品定位

专注于 Markdown 内容呈现，界面干净清爽，无广告无干扰，让你沉浸在纯粹的阅读体验中。

---

## ✨ 核心亮点

### 1. 极简设计，纯粹阅读

- 界面干净清爽，无广告无干扰
- 专注于 Markdown 内容呈现
- 支持标题、列表、代码块、表格、引用等常见格式的美观渲染

### 2. 多种打开方式

- 📁 从文件管理器直接打开
- 💬 从微信、QQ 等应用分享接收
- 📂 App 内选择文件打开

### 3. 剪贴板智能识别

- 复制 Markdown 内容后切换到 App，自动识别并提示保存
- 支持识别 Markdown 特征或长文本（50字以上）

### 4. 文件来源追溯

- 自动记录文件来源（微信、QQ、剪贴板、本地等）
- 方便文件管理和溯源

### 5. 外部文件自动保存

- 从微信、QQ 等分享的文件自动保存到本地
- 解决外部应用权限失效问题

### 6. 历史记录管理

- 快速访问最近打开的文件
- 长按菜单：分享、重命名、删除
- 自动记录阅读进度

### 7. 阅读体验优化

- 双击标题栏回到顶部
- 滚动条指示阅读位置
- 表格横向滚动支持

---

## 🎯 适用场景

- 📝 笔记内容阅读
