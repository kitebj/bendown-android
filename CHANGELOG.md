# 更新日志

本文档记录 BenDown Android 项目的版本更新历史。

**版本号规则**：`0.次版本.修订号`
- 主版本保持 0 不动
- 次版本：新功能、较大改进
- 修订号：Bug修复、小优化、UI调整

---

## [0.2.1] - 2026-03-31

### 新增
- 🍔 **顶部菜单** - 添加汉堡菜单，包含打开文件、设置、关于选项
- 🖼️ **首页空状态** - 无历史记录时显示 Logo 图片和打开文件按钮

### 优化
- 🎨 **首页布局优化** - 有历史记录时只显示列表，打开文件入口移至菜单

---

## [0.2.0] - 2026-03-31

### 新增
- 📍 **阅读位置记忆** - 打开历史文件自动定位到上次阅读位置
- 📦 **历史记录扩容** - 存储最近100个文件，显示最新20个

---

## [0.1.2] - 2026-03-31

### 新增
- 📊 **表格支持** - 支持渲染 Markdown 表格，包含对齐方式
- 📊 **底部状态条** - 显示文件大小和阅读进度百分比

### 优化
- 🎨 状态条背景色与标题栏一致
- 📜 阅读页面改用 LazyColumn 提升滚动性能

---

## [0.1.1] - 2026-03-31

### 优化
- 🎨 **历史记录布局** - 更紧凑的列表项设计

---

## [0.1.0] - 2026-03-31

### 新增
- 📁 **文件选择器** - 支持 SAF 选择本地 Markdown 文件
- 📖 **VIEW Intent** - 支持从其他应用打开 .md 文件
- 📤 **SEND Intent** - 支持接收分享的文件
- 🕐 **历史记录** - 显示最近打开的 20 个文件（紧凑布局）
- 🛡️ **安全检查** - 文件大小限制（10MB）、文件类型检查、二进制文件检测

### 重构
- 🏗️ 代码结构分层：data / ui / parser
- 📦 Markdown 解析器模块化
- 🎨 UI 组件独立封装

---

## [0.0.1] - 2024-03-30

### 新增
- 项目初始化
- 基础 Markdown 渲染功能
