# 更新日志

本文档记录 BenDown Android 项目的版本更新历史。

**版本号规则**：`0.次版本.修订号`
- 主版本保持 0 不动
- 次版本：新功能、较大改进
- 修订号：Bug修复、小优化、UI调整

---

## [0.2.7] - 2026-03-31

### 优化
- 📦 **代码重构** - 提取 formatFileSize 为公共扩展函数
- 🎨 **资源整理** - 提取颜色到 Color.kt，字符串到 strings.xml
- 🧹 **代码清理** - 移除重复代码

---

## [0.2.6] - 2026-03-31

### 修复
- 🔒 **移除多余权限** - 删除 MANAGE_EXTERNAL_STORAGE 等权限，符合 SAF 方案设计
- 🧹 **清理废弃代码** - 删除旧的 com.bendown.markdownreader 包
- 🛡️ **递归深度限制** - MarkdownParser 添加最大递归深度，防止栈溢出
- 📱 **API 兼容性** - 修复 getParcelableExtra 废弃 API，兼容 Android 14+
- 💬 **错误反馈** - 打开文件失败时显示 Toast 提示

### 优化
- ⚡ **解析缓存** - MarkdownParser 添加 LRU 缓存，提升渲染性能

---

## [0.2.5] - 2026-03-31

### 新增
- 📜 **滚动条** - 滚动时显示位置指示器，停止后自动隐藏
- ⬆️ **双击回顶** - 双击顶部标题栏快速回到顶部
- 📝 **文字选择** - 长按可选择文字，支持复制

---

## [0.2.4] - 2026-03-31

### 新增
- 📤 **阅读页菜单** - 三点菜单，包含分享和关闭功能
- 🎨 **菜单图标** - 所有菜单项添加 emoji 图标

### 优化
- 📋 菜单样式优化，图标和文字间距更紧凑

---

## [0.2.3] - 2026-03-31

### 新增
- 🗑️ **清除记录** - 菜单中新增清除历史记录功能（仅有记录时显示）
- 🍔 **顶部菜单** - 汉堡菜单，包含打开文件、清除记录、设置、关于选项

### 优化
- 🎨 **首页重新设计** - 优雅的欢迎页面（图标 + 欢迎语 + 打开文件按钮）
- 📱 **APP名称** - 桌面"笨蛋阅读器"，顶部"📚 BenDown - 笨蛋阅读器"
- 🖼️ **状态栏** - 图标改为深色，浅色背景更清晰
- 📊 **进度显示** - 阅读进度显示两位小数
- ⬅️ **后退键** - 阅读页面按后退键返回首页

## [0.1.0] - 2026-03-31

### 新增
- 📁 **文件选择器** - 支持 SAF 选择本地 Markdown 文件
- 📖 **VIEW Intent** - 支持从其他应用打开 .md 文件
- 📤 **SEND Intent** - 支持接收分享的文件
- 🕐 **历史记录** - 显示最近打开的 20 个文件（紧凑布局）
- 🛡️ **安全检查** - 文件大小限制（10MB）、文件类型检查、二进制文件检测

### 重构
- 🏗️ 代码结构分层：data / ui / parser
- 📦 Markdown 解析器模块化
- 🎨 UI 组件独立封装
