# 欢迎使用Markdown阅读器

这是一个功能强大的Markdown阅读器应用，专为阅读和编辑Markdown文档而设计。

---

## 功能特性

### 核心功能
- ✅ **文件浏览** - 快速找到并打开Markdown文件
- ✅ **Markdown解析** - 支持标准Markdown语法
- ✅ **美观渲染** - Material Design风格的阅读体验
- ✅ **文件切换** - 在多个文档间快速切换

### 支持的Markdown格式

#### 文本格式化
- **粗体文字** - 使用 `**粗体**` 或 `__粗体__`
- *斜体文字* - 使用 `*斜体*` 或 `_斜体_`
- `行内代码` - 使用 `` `代码` ``
- ~~删除线~~ - 使用 `~~删除线~~`

#### 列表样式

**无序列表：**
- 第一项内容
- 第二项内容
  - 嵌套子项1
  - 嵌套子项2
- 第三项内容

**有序列表：**
1. 第一步操作
2. 第二步操作
   1. 子步骤A
   2. 子步骤B
3. 第三步操作

**任务列表：**
- [x] 已完成的任务
- [ ] 待完成的任务
- [ ] 另一个待办项
- [x] 已完成的重要工作
  - [x] 子任务1
  - [ ] 子任务2

#### 引用块

> 这是一个引用块，可以用来强调重要内容。
> 
> 支持多行引用，每一行都以 `>` 开头。
> 
> > 引用还可以嵌套使用。

## 使用指南

### 快速开始

1. 打开应用后，您会看到文件列表
2. 点击任意文件即可查看内容
3. 使用返回按钮回到文件列表

### 阅读技巧

- 支持上下滚动查看长文档
- 标题使用不同颜色和大小，便于浏览
- 代码块使用深色背景，便于阅读
